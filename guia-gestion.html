Manual de Gestión Web

Guía paso a paso para actualizar productos y precios desde cualquier ordenador de la empresa.

1

Acceso al Panel login

Desde cualquier PC, abre el navegador y escribe la dirección de gestión:

multiserviciosborgil.com/admin.html
2

La Llave de Fotos key

Cada vez que uses un **ordenador nuevo**, el panel te pedirá tu "API Key" de ImgBB para poder subir fotos. Tenla siempre a mano.

"Es el código de números y letras que generamos al principio del proyecto."

3

Gestión de Productos inventory_2

  • Rellena los datos (Marca, Modelo, Precio, etc.).
  • Sube la foto del equipo (el sistema la recortará automáticamente).
  • Pulsa en "Añadir Producto".
  • Cuando termines todos los cambios, pulsa el botón naranja "Descargar base de datos actualizada".
4

Lanzar Cambios a la Web publish

Como no usamos base de datos de pago, tenemos que subir el archivo descargado manualmente a GitHub:

1. Entra en tu cuenta de GitHub.

2. Pulsa en "Add file" y luego en "Upload files".

3. Arrastra el archivo productos_db.js que has descargado.

4. Pulsa abajo en el botón verde Commit Changes.

¡Listo! En 1 minuto se actualizarán todos los ordenadores y móviles que visiten la web.
